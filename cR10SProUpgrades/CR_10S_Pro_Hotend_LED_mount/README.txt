# Summary

This is a mount I made to attatch a piece of led strip to my hot end. It just snaps on. You need to feed the LEDs with 12 volt which requires using a buck converter somewhere. I converted my fan to 12v already so I tied it into the fan power. 

LED strip   https://tinyurl.com/yaqrwz3d  only uses one 3 LED segment

Buck converter    https://tinyurl.com/ybjlf48j

EDIT: just found these 24v led strips so you could do it without the buck converters

24v led https://tinyurl.com/y9pctdtc

I also just uploaded another version thats sits farther back allowing a better view of the nozzle.
